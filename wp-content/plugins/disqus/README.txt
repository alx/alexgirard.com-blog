Disqus WordPress Plugin
Version 2.0-2477 (August 12, 2008)
http://disqus.com/

* Installation

 - NOTE: It is recommended that you backup your database before installing
         the plugin.

 - Unpack archive to this archive to the 'wp-content/plugins/' directory inside
   of WordPress

    - Maintain the directory structure of the archive (all extracted files
      should exist in 'wp-content/plugins/disqus/'

 - From your blog administration, click on Comments to change settings
   (WordPress 2.0 users can find the settings under Options > Disqus.)
